# Download

Latest version: [0.2.0](https://github.com/Naratzul/vellum/releases/tag/v0.2.0)

You can get the compiler from either:

- **[Nexus Mods — Skyrim Special Edition](https://www.nexusmods.com/skyrimspecialedition/mods/176581)** — Vellum mod page
- **[v0.2.0 release on GitHub](https://github.com/Naratzul/vellum/releases/tag/v0.2.0)** — under **Assets**, download the **ZIP**; it contains the `vellum.exe` binary (extract it before use).
